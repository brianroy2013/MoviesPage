The "entertainment_center.py" program creates a static html page that displays links to differnt movies.  This information for these movies are
also stored in the entertainment_center.py file.  The entertainment_center.py program uses the Move class (stored in "media.py") to store data.

Finaly the information stored in the movie class is passed to the fresh_tomatoes.open_movies_page() function (provided by Udacity) and creates the static web page ("fresh_tomatoes.html")
